## 📌 Proyecto destacado

### [Detección de Riesgo de Fraude](https://github.com/Ignadem/IgnacioDemdemian_ProyectoFinal_4Geek.git)

Proyecto final de 4Geeks Academy orientado a la detección de fraude mediante Machine Learning.

- Integré limpieza de datos, análisis exploratorio, entrenamiento, evaluación y despliegue funcional.
- Comparé modelos de Regresión Logística, Random Forest y XGBoost para evaluar desempeño predictivo.
- Construí una aplicación capaz de estimar probabilidades de fraude a partir de variables financieras clave.
- Consolidé conceptos de EDA, modelado supervisado, optimización, métricas de evaluación y visualización de resultados.
